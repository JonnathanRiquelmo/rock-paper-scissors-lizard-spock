<h2 id="about-the-project"> :pencil: About The Project</h2>

<p align="justify">
<a href="https://www.youtube.com/watch?v=Kov2G0GouBw" target=_blank>Rock, Paper, Scissors, Lizard, Spock is a game</a> of chance that expands. It is first used to settle a dispute about what to watch on TV between Sheldon and Raj in "The Lizard-Spock Expansion" at The Big Bang Theory tv show. Based on a <a href="https://zerotomastery.io/" target="_blank">ZTM</a> JavaScript course.
</p>
